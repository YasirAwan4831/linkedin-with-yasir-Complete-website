# 🧠 1. Technical SEO Foundation

## ✅ Sitemap Optimization

A fully structured sitemap is created at:

```
/public/sitemap.xml
```

Includes:

* Core pages (Home, About, Services, Pricing, Contact)
* Future blog structure (`/blog/...`)
* High-conversion landing pages

👉 Helps Google crawl and index all pages efficiently.

---

## ✅ Robots.txt Strategy

Located at:

```
/public/robots.txt
```

Configured to:

* Allow:

  * Googlebot
  * Bingbot
* Block:

  * AhrefsBot
  * SemrushBot
  * Low-quality scrapers

👉 Protects content while guiding search engines correctly.

---

## ✅ Dynamic SEO System (SPA Optimization)

A custom `<SEO />` React component is implemented.

It dynamically injects:

* Meta Titles
* Meta Descriptions
* Keywords
* Open Graph (OG) Tags
* Twitter Cards

👉 Ensures proper indexing of a **Single Page Application (SPA)**.
